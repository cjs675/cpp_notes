### notes on arrays 
- collection of elements of __same__ type 
-  can store more than 1 variable 
- each element can be accessed using an __index__ 
	- indexing begins from 0 
- pointers __point__ to address of an element/variable 
- __CANNOT__ be expanded/changed, are __fixed__ 
	- unlike vectors, which can be extended 
- syntax: 
```c++ 
int homeAddress[size];

int numbers[5] // an array of 5 integers 
```
- can be initialized in different ways: 
```c++ 
int arr1[5] = {1, 2, 3, 4, 5}; // full ionitialization

int arr2[5] = {10, 20};		// partially initialized, rest are 0

int arr[] = {7, 8, 9};		// compiler decides size = 3
```


#### accessing arrays 
- use __arrayName[index]__ 
- index begins at 0 & ends at __size - 1__ 
```c++ 
int arr[5] = {10, 20, 30, 40, 50};

cout << arr[0]; // prints 10
cout << arr[4]; // prints 50

arr[2] = 99; 		// modifies 3rd element 
```
- in C++, each element in an array is associated w/ a number 
	- # known as __array index__ 
	- can access elements of an array by using these indices 
```c++ 
// declare & initialize an array 
int x[6] = {19, 10, 8};
// array members: {19, 10, 8, 0, 0, 0} 
```

#### array traversal 
- using loop to access elements: 
```c++ 
#include <iostream> 

using namespace std;

int main() {
	int arr[5] = {10, 20, 30, 40, 50};

for (int i = 0; i < 5; i++) {
	cout << "Element at index " << i << " = " arr[i] << endl;
	}
return 0;
}		
```

### 2d array
- often defined as array of arrays 
	- aka __matrix__ 
	- can be depicted as a table of rows & columns 
- syntax: 
```c++
int arr[max_rows][max_columns];
```

#### accessing data in 2d array
- in 1D array, data can be accessed using only an index 
- in 2D array, we can access cells individually by using indices of cells 
	- 2 indices attached to a single cell: 
		- index 1: row number 
		- index 2: column number 
- syntax to store value stored in any particular cell of an array: 
```c++ 
int x = a[i][j]; 
	// i = row index 
	// j = column index 
```

#### initializing 2d array
- when declaring a 1D array, no need to specify __size__ 
- for 2D
	- need to specify at least row size 

```c++ 
// syntax to declare 2d array 

int arr[2][2] = {1, 2, 3, 4};
```

- **number of elements** present in a 2d array will __always__ equal: 
(__# of rows__ * __# of columns__) 

#### multidimensional arrays 
```c++ 
int x[3][4];

// can hold max of 12 elements 
```

#### multidim. array initialization
```c++ 

int test[2][3]= { {2, 4, 5}, {9, 0, 19} };
```


#### advantages of array 
- represent multiple data elements of same type using single name
- accessing/searching an element in an array is easy by using index number 
- can be traversed easily just be incrementing index by 1
- allocate memory in contiguous memory locations for all its data elements

#### drawbacks of arrays 
- cannot be resized 
	- made larger or smaller 
- when we wish to alter, we copy the elements & move to a new array 




#### OOP arrays 
- we can create object as arrays 
```c++ 
int main() {
	Employee emp[30];
}
```

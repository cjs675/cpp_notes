# AssignMate 

### Purpose: Design a minimal, GUI-based assignment tracker 

### Framework: QT GUI 

### Features:  

- input classes 
    - input assignments from respective classes 
    - include: 
        - due dates 
        - mark as complete 
        - sort: 
            - past vs. present 
        - assignments completed vs. to complete 
- store user data locally (SQLITE)

### if vs switch-case-break 
- switch: only char & integer work 
- break: stopping of code & continuing on with next case i

- for loop using arrays 

```
int main() {
	int numbers[5] = {2, 3, 3, 2, 5}, sum = 0;

	for (int i = 0; i < 5; i++) {
		sum += numbers[i]; // sum=sum+numbers[i]
	}
	cout << "The total is " << sum << endl;
}
```

# 07/13 Class Notes - Pointers


- **Pointer:**  variable that stores **memory address** of another variable 
	- instead of holding the data directly, it points to the location where the data is stored 
	- powerful, can be challenging to use 
	- use correctly, responsibly, when appropriate 
	- used in dynamic data structures, creation & manipulation 
		- i.e, linked lists, queues, stacks trees 
- **Pointer Types** 
	```cpp
	int *x; // pointer to integer 
	double *x; // pointer to double float 
	char *ch // pointer to a character 
	```
- **Pointer Declaration** 
	- Declaration: 
	```cpp
	int *countPtr, count
	``` 
	- **countPtr** is of type int * (i.e., a pointer to an int) 
	- count is an int, not a pointer to an int 
	- each variable being declares as a pointer must be preceded by __*__ 
	- pointers can point to objects of any data type  

- **Important Operators:** 
	```cpp
	// Sample code 
	int x = 10;
	int *ptr; // pointer declaration 
	ptr = &x; // assign address of x to ptr
	```
	- **address-of operator (&):** gives memory address of a variable 
	```cpp
	cout << &x; // prints address of x
	```
	- **dereference operator (*):** accesses value at address stored in the pointer 
		- may also be used on the _left_ side of an assignment statement, or to receive an input value  
	```cpp
	cout << *ptr; // prints value of x (10) 
	```
- **Pointer initialization:** should be initialized with **null** ptr or an address of the corresponding type (when declared or in an assignment) 
	- **null pointer:** pointer that points to "nothing" 
	- 0 = only integer that can be assigned to a pointer 


- 3 ways to pass args to a function (in C++): 
	1. pass-by-value
	2. pass-by-reference
	3. pass-by-reference with pointer args 

**Pass-by-Reference with Pointers:** 
- when calling a function with an arg that should be modified, the address of the argument is passed to the called function 
	- apply __&__ to name of variable with contents to be modified 
	- use __*__ to dereference address & modify the contents 
- a function receiving an address as an argument __must__ have a pointer as its corresponding parameter 


**Pointer Expressions & Pointer Arithmetic** 
- arithmetic ops may be performed on pointers 
	- appropriate __only__ for pointers to built-in array elements 
	- **++** incrementing a pointer 
	- **--** decrementing a pointer 
	- __+ or +=__ adding an integer to a pointer 
	- __- or -=__ subtracting an integer from a pointer 
	- subtracting one pointer from another (appropriate only for two pointers that point to elements of the same built-in array 

**Relationship between Pointers & Built-in Arrays**
- pointers that point to built-in arrays can be subscripted exactly as built-in array names can 
	- pointer/offset notation, if the pointer points to 1st element of a builtin array, the offset = same as an array subscript 
	- all subscripted array expressions can be written with a pointer and an offset, using either the built-in array's name as a pointer or using a separate pointer that points to the built-in array 
 
**String Literals** 
- may initialize built-in array of chars or type const char * variable 
```cpp
char color[] = "blue"; // creates 5-element built-in array containing chars: 'b','l','u','e'

const char *colorPtr = "blue"; // creates pointer var colorPtr that points to letter b in string "blue" which ends in '\0'
```


**Pointer-based strings, I/0** 
- a string can be read into a built-in array of chars using **cin** (until whitespace char or EOF is encountered) 
```cpp
char word[20];
```
- **setw** can be used to ensure that string read into an array of chars doesn't exceed size of the built-in array 
```cpp
cin >> word;
cin >> setw( 20 ) >> word;
```
- in member function **getline** can input entire line of text, using 3 args: 
	- built in array of chars in which text can be stored 
	- length 
	- delimiter character ('\n' by default) 
	- built in array of chars representing a null-terminated string can be ouput with **cout** and **<<**
		- chars are output until \0 is encountered 
```cpp
char sentence[80];
cin.getline(sentence, 80, '\n');
cin.getline(sentence, 80);

cout << sentence;
```

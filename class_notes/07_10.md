### vectors 
- sequential based container 
- can only be implemented dynamically 
- have dynamic size (can resize themselves) 
- occupy more memory than arrays 
- accessing elements takes more time than accessing elements of array
- are __NOT__ index based data structures 
- only available in C++ 
- auto deallocated from heap

#### basic ops 
- vector class provides methods to perform different ops: 
	- add elements 
	- access elements 
	- change elements 
	- remove elements 
	- get vector size
	- capacity
	- max_size
	- resize 
	- empty
